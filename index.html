About Me

Hi, I’m Sakib Hossain Computer Science & Technology student who writes code that doesn’t just live on a screen. I believe the best software exists where logic meets physics, so I build systems that move motors, read sensors, and react to the real world.

I sharpen my logical reasoning through problem solving on LeetCode, using Python for rapid prototyping and C when I need fine-grained control over memory and performance. On the web, I craft interactive dashboards with HTML, CSS, and JavaScript to visualize sensor data and command my robots in real time.

What excites me most is tying everything together. I design the back‑end logic in Python, wrap it in a JavaScript interface, and implement the low‑level actuators in C. Understanding both web protocols and serial communication lets me create robots that are just as easy to control from a browser as they are from a terminal.

I’m actively looking for internships, open‑source collaborations, and hackathons where software meets hardware. I’m also eager to dive deeper into ROS 2 and Embedded Linux. Whether it’s optimizing an algorithm to beat the 99th percentile or tuning a PID controller to keep a robot balanced, I’m obsessed with making things work smarter. Let’s build something that doesn’t just compute — but acts.
